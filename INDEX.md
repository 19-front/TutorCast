## 🔍 Key Concepts

### Label Engine
The main processor that:
- Observes raw events from KeyMouseMonitor
- Matches events against active profile's mappings
- Publishes semantic labels (e.g., "PAN") with colors
- Auto-clears after 1.5 seconds

**File**: [Models/LabelEngine.swift](Models/LabelEngine.swift)

### Profile System
Manages:
- Action-to-label mappings
- Multiple profiles (AutoCAD, Photoshop, Default)
- Profile persistence and selection
- Active profile tracking

**Files**: [Models/Profile.swift](Models/Profile.swift), [Models/SettingsStore.swift](Models/SettingsStore.swift)

### Semantic Colors
Maps labels to colors:
- Orange: Navigation (PAN, ORBIT)
- Cyan: Zoom (ZOOM IN, OUT)
- Green: Select (SELECT, PICK)
- Red: Destructive (DELETE, CUT)
- White: Default

**Code**: In [Models/LabelEngine.swift](Models/LabelEngine.swift) → `colorForLabel()`

### Menu Bar Switcher
Quick-access profile menu:
- Shows current profile name
- Lists all available profiles
- Instant profile switching
- Checkmark on active profile

**Code**: In [TutorCastApp.swift](TutorCastApp.swift) → `MenuBarContentView`
